Home Network Upgrade

In this project, I analyze my home network for expansion oppurtunities and ultimately upgrade it in a way to give me more room to expand in the future.

Requirements

I originally started this project because the Wi-Fi in my house was terrible, so I first set a requirement to improve Wi-Fi coverage in every corner of my house. The only network switches in my network was a 4 port one on the router and another one with a maximum bandwidth of 100Mbps while my Internet speed was higher than that. From this, I figured that I needed to get two wireless access points and some network switches. In my research, I also came across Pi-Hole, a DNS sinkhole to prevent advertisements network-wide. I decided to grab a Raspberry Pi to try it out.

UniFi

After doing research on different wireless access points, I decided to choose Ubiquiti's UniFi Access Point. UniFi is an enterprise-like Wi-Fi system that is scaleable and easily manageable. It connects to a UniFi Controller which manages any access point I adopt on it. It is also able to manage a UniFi Security Gateway, Ubiquiti's enterprise gateway router. I choose the UAP as my wireless access points because of it's guest network policies that I can control such as restricting subnets and download speed.

Process

First, I had to analyze which areas of my house had the least Wi-Fi coverage and which ones would need the most. I decided to place one access point in my game room on the second floor because it had many devices and it was near my room, and the other on the opposite of my house in the study room. Next I had to terminiate my house's existing cat5e wires into a network switch which would connect to my router. That network switch was dedicated to my house's ports, while another switch was dedicated to the desktops and access points. After I had all the physical networking done, I worked on configuring the Wi-Fi. I analyzed which channels were the least congested, and set my access points appropriately. I also set up guest policies for my friends or relatives that come over, so they have access to the internet but not access to anything else on the network. I ran a speed test in my room and my new speeds were more than 5 times faster. I then set up a Pi-hole to filter out advertisements for my smart TV in my gameroom, and on an average day, around 15-20% requests are blocked

Future Upgrades

In the future, I hope to replace my current router so I can get more familiar with configuring different subnets for different tasks. I want to get a UniFi Security Gateway because that will be easily configurable from my UniFi controller that is already managing my access points. I also want to get a server to add to my network to centralize my files, create a VPN endpoint, and just play around with different technologies
